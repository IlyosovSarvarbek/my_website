## Features

- ✅ Organized question bank for Part 1 / Part 2 / Part 3  
- ✅ High-quality sample answers (band 7+) with clear structure and useful phrases  
- ✅ Browse, search, and filter questions by topic and section  
- ✅ Bookmark questions and sample answers for quick review  
- ✅ Lightweight, offline-friendly content (no heavy server dependency)  
- ✅ Simple, responsive UI optimized for mobile screens

## Tech stack

- React Native (Expo)  
- JavaScript · Expo CLI · AsyncStorage (local persistence)

## Quick start

```bash
git clone https://github.com/IlyosovSarvarbek/ielts_speaking_learning_app
cd ielts_speaking_learning_app
yarn install
expo start
# then open on a device/emulator via the Expo app
```

## UX highlights

- Focused learning flow: pick a section → choose a question → read model answer  
- Clear answer breakdown: introduction, main points, concluding sentence  
- Fast navigation between related topics for targeted practice

## Tips for using the app

- Use bookmarks to build a personalized revision list  
- Practice Part 2 prompts aloud, then compare with model answers  
- Mix randomized questions for exam-style timed practice
